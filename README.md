Q&A：
1. 什么是Agent 的流式返回会更偏运行时事件结构？流式放回的类型？
    流式返回有三种类型： `message` | `text` | `json`
    结构型通常指json这种格式，常用于使用工具后得到的数据，可以直接用于分析以及数据处理。
    三者的使用场景：
    message --> 常用于有交互体验的场景，比如聊天窗口逐字输出、AI 助手回复、多轮对话。
    text ---> 常用于在控制台中中输出，直接拿到所有的结果，适用场景：cli 、日志流。
    json ---> 需要边生成边解析结构数据，比如AI 输出固定 schema（比如工具调用）。
